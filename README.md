# ⚽ Soccer Performance Analytics

## Project Overview
An end-to-end data analytics project analysing European football
performance across 11 leagues, 25,979 matches and 11,060 players
from 2008 to 2016.

## Tools Used
- SQL (DB Browser for SQLite)
- Python / Pandas (coming soon)
- Power BI Dashboard (coming soon)

## Business Questions Answered
1. Which teams win the most at home?
2. Which league produces the most goals per match?
3. Which teams are the most attacking at home?
4. How clean and complete is the dataset?

## Key Findings
- FC Barcelona and Real Madrid dominate home performance
- Netherlands Eredivisie is the highest scoring league (3.08 goals/match)
- Real Madrid averages 3.32 goals per home match
- Dataset is high quality with zero missing values across all key columns

## Project Structure
- queries/ — SQL analysis files
- data_quality_checks.sql — Data cleaning report

## Dataset
European Soccer Database — Kaggle
25,979 matches across 11 European leagues
